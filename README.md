# IBM-HR-Analytics-Employee-Attrition-and-Performance
Predicting Employee Attrition using R

The data set which is used here is IBM HR Analytics Attrition data set from kaggle.com. URL: https://www.kaggle.com/pavansubhasht/ibm-hr-analytics-attrition-dataset
Tools & Techniques Usng RStudio, I have built classification model to predict which employee is likely to churn and help the business to device policies and attract back the right talent.
